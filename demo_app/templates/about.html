About This Project

Research Context

This web application is a demonstration component of an MSc thesis investigating the use of classical machine learning techniques for detecting acoustic markers associated with Parkinson's Disease in voice recordings.

Methodology

The system extracts acoustic features from voice recordings, including:

  • Prosodic features (fundamental frequency, jitter, shimmer)
  • Spectral features (MFCCs, spectral centroid, bandwidth)
  • Voice quality measures (harmonic-to-noise ratio)

These features are processed by a Random Forest classifier trained on the MDVR-KCL dataset using grouped cross-validation to prevent data leakage.

Limitations

  • Small training dataset (~37 subjects)
  • Single speech task (ReadText)
  • No external validation on independent cohorts
  • Recording conditions may affect results

⚠️ Important Disclaimer

This application is a research demonstration developed as part of an MSc thesis. It is NOT intended for medical diagnosis or clinical use.

Parkinson's Disease is a complex neurological condition that can only be diagnosed by qualified medical professionals through comprehensive clinical assessment.

The predictions made by this system:

  • Are based on limited training data
  • Have not been clinically validated
  • Should not influence medical decisions
  • May produce incorrect results

Technical Information

Dataset:
MDVR-KCL (Mobile Device Voice Recordings)
Model Type:
Random Forest Classifier
Features:
47 acoustic features (baseline set)
Validation:
Grouped Stratified 5-Fold CV
